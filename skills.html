SKILLS

  • JAVA
    • Java is a versatile, object oriented programming language known for its platform independence. Developers by Sun Microsystems(now owned by Oracle), it features a rich ecosystem, including the Java Virtual Machine(JVM), extensive libraries and widespread use in web applications, making it a popular choice for developers.


  • SQL
    • Structured Query language is a powerful domain-specific language used for managing and manipulating relational database.It enables users to perform tasks like data retrivel, insertion, deletion and modification making it crucial tool for data management and analysis in various applications from web development to business intelligence.

  • HTML
    • (Hypertext Markup Language) is the standard markup language used to create web pages and structure their content. It consists of elements, tags and attributes that define the structure and presentation of text, images, links, and multimedia on the internet. Browsers interpret HTML to render web content and enable user interaction.

  • CSS
    • Cascading Style Sheets is a language used for describing the presentation and layout of web pages. It controls the visual aspects of HTML elements including fonts, colors, spacing and positioning, allowing web designers and developers to create attractive and responsive user interfaces for websites and web applications.

  • Java Script
    • JavaScript is a versatile and widely used programminglanguage for web development. It enables interactive and dynamic features in web pages, enhancing user experiences. With its client-side capabilities, JavaScript runs directly in browsers, allowing for tasks like form validation, animations and AJAX requests. It's a fundamental tool for modern web applications.
